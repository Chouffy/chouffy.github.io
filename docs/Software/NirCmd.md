Is a [[Software]] that is part of [[NirSoft]] package which enable to do multiple tasks without a [[GUI]]
## Link
- [Website](http://www.nirsoft.net/utils/nircmd.html)
## Set the display to a specific resolution
- Usage: `setdisplay {monitor:index/name} [width] [height] [color bits] {refresh rate} {-updatereg} {-allusers}`
- [Documentation](http://nircmd.nirsoft.net/setdisplay.html)
- Example:
	- `nircmd.exe setdisplay 1920 1080 32 60` for a 1920x1080 resolution with 32 bits and 60 Hz
